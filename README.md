📖 Overview

MedSOS is a cross-platform healthcare application built with Flutter and Firebase. It helps users manage medications, receive timely reminders, access emergency assistance, and stay connected with trusted family members or caregivers.

Our mission is to make healthcare management simpler, safer, and more accessible for everyone.

---

✨ Features

- 💊 Medication management
- ⏰ Smart medication reminders
- 🗣️ Voice Assistant (Speech-to-Text & Text-to-Speech)
- 📍 Emergency SOS with live location
- 👨‍👩‍👧 Emergency contacts
- 🔔 Local notifications
- ☁️ Firebase authentication & cloud database
- 🌐 Cross-platform support (Android, iOS, and Web)
